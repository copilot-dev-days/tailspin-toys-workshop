# Tailspin Toys

Este repositorio contiene el proyecto para un taller guiado para explorar GitHub Copilot CLI. El proyecto es un sitio web para una empresa ficticia de financiamiento colectivo de juegos, con un backend [Flask](https://flask.palletsprojects.com/en/stable/) usando [SQLAlchemy](https://www.sqlalchemy.org/) y frontend [Astro](https://astro.build/) usando [Svelte](https://svelte.dev/) para páginas dinámicas.

## Iniciar el taller

**Para comenzar el taller, empieza en [workshop/es/README.md](./workshop/es/README.md)**

O, si solo quieres ejecutar la aplicación...

## Iniciar el sitio

Se ha creado un archivo de script para iniciar el sitio. Puedes ejecutarlo con:

```bash
# Linux/macOS/Codespaces
./scripts/start-app.sh

# Windows (PowerShell)
.\scripts\start-app.ps1
```

¡Luego navega al [sitio](http://localhost:4321) para ver el resultado!
