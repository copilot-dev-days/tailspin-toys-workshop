# Tailspin Toys

Este repositório contém o projeto para um workshop guiado para explorar o GitHub Copilot CLI. O projeto é um site para uma empresa fictícia de financiamento coletivo de jogos, com um backend [Flask](https://flask.palletsprojects.com/en/stable/) usando [SQLAlchemy](https://www.sqlalchemy.org/) e frontend [Astro](https://astro.build/) usando [Svelte](https://svelte.dev/) para páginas dinâmicas.

## Iniciar o workshop

**Para começar o workshop, inicie em [workshop/pt_BR/README.md](./workshop/pt_BR/README.md)**

Ou, se você só quer rodar o aplicativo...

## Iniciar o site

Um arquivo de script foi criado para iniciar o site. Você pode executá-lo com:

```bash
# Linux/macOS/Codespaces
./scripts/start-app.sh

# Windows (PowerShell)
.\scripts\start-app.ps1
```

Depois navegue até o [site](http://localhost:4321) para ver o resultado!
